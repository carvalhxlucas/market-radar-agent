# MarketRadar Frontend

Interface web moderna para o agente MarketRadar, desenvolvida em React com TypeScript.

## Instalação

```bash
cd frontend
npm install
```

## Execução

```bash
npm start
```

O frontend estará disponível em `http://localhost:3000`.

Certifique-se de que a API backend está rodando em `http://localhost:8000`.

## Tecnologias

- React 18
- TypeScript 5
- WebSocket para comunicação em tempo real

## Build para Produção

```bash
npm run build
```
